About This Project

About the project

This web map app helps users explore riparian ecosystem patterns in the Middle Rio Grande study area. Users can click polygons to view vegetation cover, area, and ET trends, and can switch the map display between vegetation community, tree cover, shrub cover, and herbaceous cover.
